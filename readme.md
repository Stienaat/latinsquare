# Latin Square – V2.1

Een 7×7 Latin Square puzzelspel met drie moeilijkheidsgraden, joker‑hints, oplossing‑weergave, taalkeuze en automatische opslag.  
Volledig client‑side, geen server nodig.

## 🎮 Functies

- 7×7 Latin Square puzzel
- 3 levels:
  - **Level 1:** rijen + kolommen
  - **Level 2:** rijen + kolommen + 2 hoofddiagonalen
  - **Level 3:** rijen + kolommen + alle 14 wrap‑diagonalen
- **Joker‑hint:** toont alle foute tegels
- **Oplossing:** toont de correcte oplossing van het huidige spel
- **Taalkeuze:** NL, FR, EN, DE
- **Autosave:** spel wordt automatisch opgeslagen bij elke zet
- **Herstel:** bij herladen of herstarten wordt het spel hervat waar je gebleven was
- **Volledig offline:** werkt in elke browser via een lokale server

## 📦 Bestanden


## ▶️ Spelen

1. Start een lokale webserver (bijv. via VS Code Live Server of `http-server`)
2. Open de browser op:  
   `http://localhost/Latinsquare`
3. Klik op twee tegels om ze te wisselen
4. Gebruik de joker als je vastzit
5. Start een nieuw spel via de toolbar

## 💾 Opslag

Het spel bewaart automatisch:

- bord
- level
- aantal zetten
- hint‑status

Bij het openen van de pagina wordt het opgeslagen spel automatisch hervat.

## 🧩 Doel van het spel

Vorm een geldige Latin Square:

- elke kleur mag **maximaal één keer** voorkomen per rij en kolom  
- afhankelijk van het level gelden extra diagonaalregels  
- het spel is **altijd oplosbaar**
